# 学习绘制 Mermaid 图表和 Draw.io 图表来丰富你的论文、文档与 PPT

:::info 课程简介
在数字化时代，优秀的图表是传达复杂信息的关键。本课程将教您掌握两种强大的图表工具——Mermaid 和 Draw.io，并学会使用 AI 辅助快速创建专业图表，让您的学术论文、技术文档和商务演示更加出色。
:::

:::tip 🎥 观看视频演示
课程概览视频已迁移到本期项目的[视频教程](../videos/)总页，方便你在一个地方找到所有教学视频。
:::

## 学习目标

完成本课程后，您将能够：

- **代码化图表**：使用 Mermaid 语法快速生成流程图、时序图、甘特图等各类图表
- **专业设计**：运用 Draw.io 创建复杂的架构图、网络拓扑图和业务流程图
- **AI 增强**：利用 Gemini CLI 自动生成和优化图表，提升创作效率
- **场景应用**：掌握图表在论文、文档、PPT 等不同场景的最佳实践

## 为什么要学习专业图表制作

- **提升信息传达效率**：一图胜千言，优秀的图表能让复杂概念一目了然，大幅提升沟通效率。
- **学术研究必备技能**：高质量的图表是学术论文的重要组成部分，直接影响论文的专业度和接受率。
- **职场竞争力加分**：能够快速制作专业图表的能力，是技术人员和管理者的核心竞争力之一。
- **AI 时代新技能**：结合 AI 工具进行图表创作，让您站在技术前沿，工作效率倍增。

## 所需工具

**必备工具**：

- 浏览器（Chrome/Firefox/Edge）
- 文本编辑器（VS Code 推荐）
- Gemini CLI（AI 辅助）

**在线工具**：

- [Mermaid Live Editor](https://mermaid.live)
- [Draw.io Online](https://app.diagrams.net)
- [GitHub（图表渲染）](https://github.com)

**可选工具**：

- Draw.io 桌面版
- Node.js（本地开发）
- 各种 IDE 插件

## 学习路径建议

<Admonition type="tip" title="推荐学习顺序">
1. **初学者路径**：Mermaid 基础 → Draw.io 基础 → 场景应用 → 实战项目
2. **快速上手路径**：AI 辅助创作 → Mermaid 基础 → 实战项目
3. **专业深化路径**：完整按顺序学习所有模块，并完成所有实战项目
</Admonition>
