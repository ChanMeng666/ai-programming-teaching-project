Node.js 是一个免费工具，让你可以在电脑上运行 Gemini CLI。

<Tabs>
  <Tab title="Windows">
    1. 前往 [nodejs.org](https://nodejs.org)
    2. 点击绿色大按钮 **LTS** 下载安装程序
    3. 打开下载的 `.msi` 文件
    4. 点击**下一步**完成所有屏幕 —— 默认设置即可
    5. 点击**安装**，然后点击**完成**
  </Tab>
  <Tab title="macOS">
    1. 前往 [nodejs.org](https://nodejs.org)
    2. 点击绿色大按钮 **LTS** 下载安装程序
    3. 打开下载的 `.pkg` 文件
    4. 点击**继续**完成所有屏幕 —— 默认设置即可
    5. 点击**安装**，然后点击**关闭**
  </Tab>
</Tabs>

<Accordion title="如何知道安装成功了？">
打开终端并输入：

```bash
node --version
```

你应该看到类似 `v22.x.x` 的版本号。如果看到错误，尝试关闭并重新打开终端，然后再次运行该命令。
</Accordion>
