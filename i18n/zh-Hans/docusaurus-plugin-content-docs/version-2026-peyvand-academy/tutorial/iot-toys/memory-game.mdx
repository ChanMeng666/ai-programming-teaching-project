<Tip>
**难度：●●○ 中等** · 搭建时间：约 3 分钟
</Tip>

难度提升：应用现在必须**记住**序列。观看越来越长的颜色与声音序列，然后用方向键重复。

## 提示词（请复制英文原文）

复制以下内容到 [claude.ai](https://claude.ai)：

```
Make a memory game like Simon Says. Show four big coloured pads for the Up, Down, Left and Right arrow keys, each with its own sound. Flash a sequence of pads that gets one longer each round, and I repeat it with the arrow keys. Show the round number, and say game over if I press the wrong one. Make it work with a normal keyboard.
```

<Tip>
四个方向键垫正好对应四根香蕉。应用现在有输赢规则——不只是对单次按键做出反应。
</Tip>

## 步骤

<Steps>
  <Step title="新开对话或继续">
    每个玩具可以用新对话，也可以在同一对话里继续。
  </Step>
  <Step title="粘贴提示词">
    复制上面的英文提示词，按 Enter 发送。
  </Step>
  <Step title="一起玩一轮">
    看序列闪烁，然后用方向键重复。多试几轮。
  </Step>
  <Step title="轮流玩">
    让朋友看序列、你来重复——然后交换角色。
  </Step>
</Steps>
