## üéµ Audio Function Implementation

### Music Player
```javascript
// Audio playback control example
const audioPlayer = {
  play: () => {
    const audio = document.getElementById('bgMusic');
    audio.play();
  },
  pause: () => {
    const audio = document.getElementById('bgMusic');
    audio.pause();
  },
  toggle: () => {
    const audio = document.getElementById('bgMusic');
    audio.paused ? audio.play() : audio.pause();
  }
};
```
