# Install Cursor Editor

## Download Cursor

<div className="download-steps">

### 1. Get the Installer
1. Visit the official website: https://cursor.sh/
2. Click the "Download" button to download the installer
3. Wait for the download to complete

### 2. Installation Process
<Tabs>
  <TabItem value="windows" label="Windows" default>
    1. Double-click the downloaded installer
    2. Click "Next" to continue installation
    3. Select installation location (recommended to keep default)
    4. Wait for installation to complete
    5. Click "Finish" to complete installation
  </TabItem>
  <TabItem value="mac" label="macOS">
    1. Open the downloaded .dmg file
    2. Drag Cursor into the Applications folder
    3. Click "Open" to confirm on first run
  </TabItem>
</Tabs>

### 3. Initial Setup
1. Launch Cursor from the Start menu or desktop
2. A welcome screen will appear
3. Choose your preferred theme (dark or light)
4. Complete initial setup

</div>

:::tip Installation Recommendations
- Recommend installing with administrator privileges
- If a firewall prompt appears during installation, please allow access
- Recommend restarting your computer after installation
:::
