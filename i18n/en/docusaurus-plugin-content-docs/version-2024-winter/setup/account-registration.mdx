# Register Necessary Accounts

## GitHub Account Registration

<div className="registration-steps">

### Registration Process
1. Visit: https://github.com/
2. Click "Sign up" in the top right corner
3. Fill in registration information:
   - Enter email address
   - Create password
   - Enter username (this will be your GitHub ID)
   - Complete email verification
4. Select the free plan (Free)
5. Complete the questionnaire verification

:::tip Registration Recommendations
- Choose a professional username
- Use a secure password
- Recommend enabling two-factor authentication
:::

</div>

## Vercel Account Registration

<div className="vercel-setup">

### Registration Steps
1. Visit: https://vercel.com/
2. Click "Sign Up"
3. Select "Continue with GitHub"
4. Authorize Vercel to access your GitHub account
5. Complete account setup

:::info Why Choose Vercel
- Free static website hosting
- Automatic deployment functionality
- Excellent performance and reliability
- Simple and easy-to-use interface
:::

</div>
