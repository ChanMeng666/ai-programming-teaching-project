# System Preparation

## Check Computer System

### System Requirements
<div className="system-check">

1. Press `Windows key + R`
2. Type `msinfo32` and press Enter
3. Check if "System Type" is 64-bit or 32-bit
4. Ensure the system is Windows 10 or higher

:::tip System Recommendations
- Recommended: 64-bit Windows 10 or Windows 11 system
- Keep the system updated with latest updates
- Ensure sufficient disk space (at least 20GB recommended)
:::

</div>

## Network Environment Preparation

### VPN Configuration
- Some websites and tools may require VPN access
- It's recommended to have a stable VPN tool prepared in advance
- Ensure you can access GitHub and Vercel websites

:::caution Notice
Please ensure you use legal and compliant network tools and comply with relevant laws and regulations.
:::
