<Tip>
**Difficulty: ●○○ Easy** · Build time: ~3 minutes
</Tip>

The warm-up: press a key, hear a note, watch a big button flash. Type this prompt into Claude.

## The prompt

Copy and paste this into [claude.ai](https://claude.ai):

```
Make a fun web-page piano. When I press Space, the arrow keys, or W A S D F G, play a different musical note and flash a big colourful button for that key. Make it work with a normal keyboard.
```

<Tip>
Stress the last sentence — **"make it work with a normal keyboard"** — because that is what lets you play it without the Makey Makey board. The piano uses Space, arrows, and W A S D F G, which map directly to Makey Makey pads.
</Tip>

## Steps

<Steps>
  <Step title="Open Claude">
    Go to [claude.ai](https://claude.ai) and start a new chat.
  </Step>
  <Step title="Paste the prompt">
    Copy the prompt above and press Enter.
  </Step>
  <Step title="Watch the Artifact build">
    Claude writes your piano in the Artifact preview panel. This may take a few seconds.
  </Step>
  <Step title="Play a few keys">
    Click into the Artifact and press Space, arrow keys, or W A S D F G. You should hear notes and see buttons flash.
  </Step>
</Steps>
