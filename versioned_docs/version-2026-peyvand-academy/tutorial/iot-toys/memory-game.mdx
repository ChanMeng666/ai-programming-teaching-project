<Tip>
**Difficulty: ●●○ Medium** · Build time: ~3 minutes
</Tip>

A step up: now the app has to **remember**. Watch a growing colour-and-sound sequence, then repeat it with the arrow keys.

## The prompt

Copy and paste this into [claude.ai](https://claude.ai):

```
Make a memory game like Simon Says. Show four big coloured pads for the Up, Down, Left and Right arrow keys, each with its own sound. Flash a sequence of pads that gets one longer each round, and I repeat it with the arrow keys. Show the round number, and say game over if I press the wrong one. Make it work with a normal keyboard.
```

<Tip>
The four arrow pads are exactly four bananas later. The app now has rules you can win or lose — not just react to a single key press.
</Tip>

## Steps

<Steps>
  <Step title="Start a new chat or continue">
    You can use a fresh Claude chat for each toy, or continue in the same one.
  </Step>
  <Step title="Paste the prompt">
    Copy the prompt above and press Enter.
  </Step>
  <Step title="Play one round together">
    Watch the sequence flash, then repeat it with the arrow keys. Try a few rounds.
  </Step>
  <Step title="Take turns">
    Let a friend watch the sequence while you repeat it — then swap roles.
  </Step>
</Steps>
