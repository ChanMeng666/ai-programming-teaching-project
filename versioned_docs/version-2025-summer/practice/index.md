# 实践项目

## 2025年夏季她行活动 - 实践学习模块

在这个模块中，参与者将通过实际项目开发来应用所学知识，提升实战技能。

## 项目类型

### 个人作品集项目
- 构建个人技术博客
- 开发展示型网站
- 创建AI应用demo

### 团队协作项目
- 开源项目贡献
- 小组项目开发
- 社区建设参与

### 创新实验项目
- AI工具创新应用
- 跨领域项目探索
- 创业想法验证
