# Draw.io 专业设计

## 🎨 Draw.io 简介

Draw.io（现称 diagrams.net）是一款功能强大的在线图表绘制工具，完全免费且无需注册。它提供了丰富的图形库和专业的绘图功能，适合创建各种复杂的专业图表。

### 核心优势

- **完全免费**：所有功能免费使用，无付费墙
- **无需注册**：打开即用，保护隐私
- **多平台支持**：Web、桌面版、VS Code 插件
- **丰富模板**：内置大量专业模板
- **多格式导出**：支持 PNG、SVG、PDF、XML 等格式
- **实时协作**：支持团队协作编辑

#### Draw.io 绘制技巧

1. **使用容器分组**
   - 将相关组件放入容器
   - 设置容器背景色区分层次
   - 启用容器折叠功能

2. **对齐和分布**
   - 使用 Arrange → Align 对齐元素
   - 使用 Distribute 均匀分布
   - 启用网格吸附

3. **连接线优化**
   - 使用直角连接线
   - 添加连接点锚定
   - 设置线条跳跃避免交叉

4. **快捷键**
   - `Ctrl+D`: 复制元素
   - `Ctrl+G`: 组合元素
   - `Ctrl+Shift+G`: 取消组合
   - `Alt+Shift+方向键`: 精确移动

</div>
