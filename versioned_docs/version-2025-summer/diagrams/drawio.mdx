# Draw.io 专业设计

## Draw.io 简介

Draw.io（现称 diagrams.net）是一款功能强大的在线图表绘制工具，完全免费且无需注册。它提供了丰富的图形库和专业的绘图功能，适合创建各种复杂的专业图表。

### 核心优势

- **完全免费**：所有功能免费使用，无付费墙
- **无需注册**：打开即用，保护隐私
- **多平台支持**：Web、桌面版、VS Code 插件
- **丰富模板**：内置大量专业模板
- **多格式导出**：支持 PNG、SVG、PDF、XML 等格式
- **实时协作**：支持团队协作编辑

## 快速开始

### 访问方式

- **在线版本**：访问 [app.diagrams.net](https://app.diagrams.net)，无需安装，云端保存
- **桌面应用**：[下载桌面版](https://github.com/jgraph/drawio-desktop/releases)，离线使用，性能更好
- **VS Code 插件**：扩展名 Draw.io Integration，集成开发环境

<TabItem value="tips" label="绘制技巧">

#### Draw.io 绘制技巧

1. **使用容器分组**
   - 将相关组件放入容器
   - 设置容器背景色区分层次
   - 启用容器折叠功能

2. **对齐和分布**
   - 使用 Arrange → Align 对齐元素
   - 使用 Distribute 均匀分布
   - 启用网格吸附

3. **连接线优化**
   - 使用直角连接线
   - 添加连接点锚定
   - 设置线条跳跃避免交叉

4. **快捷键**
   - `Ctrl+D`: 复制元素
   - `Ctrl+G`: 组合元素
   - `Ctrl+Shift+G`: 取消组合
   - `Alt+Shift+方向键`: 精确移动

</TabItem>

## UML 图表

### 用例图（Use Case Diagram）

**绘制步骤**：

1. **添加参与者**：从 UML 图形库拖入 Actor
2. **创建用例**：使用椭圆形表示用例
3. **建立关联**：连接参与者和用例
4. **添加关系**：include、extend、generalization

### 活动图（Activity Diagram）

**关键元素**：

- **开始节点**：实心圆
- **活动节点**：圆角矩形
- **判断节点**：菱形
- **并行处理**：分叉/汇合条
- **结束节点**：双圆圈
