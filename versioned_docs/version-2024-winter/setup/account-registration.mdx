# 注册必要账号

## GitHub账号注册

<div className="registration-steps">

### 注册流程
1. 访问：https://github.com/
2. 点击右上角"Sign up"
3. 填写注册信息：
   - 输入邮箱地址
   - 创建密码
   - 输入用户名（这将是你的GitHub ID）
   - 完成邮箱验证
4. 选择免费计划（Free）
5. 完成问答验证

:::tip 注册建议
- 选择一个专业的用户名
- 使用安全的密码
- 建议开启两步验证
:::

</div>

## Vercel账号注册

<div className="vercel-setup">

### 注册步骤
1. 访问：https://vercel.com/
2. 点击"Sign Up"
3. 选择"Continue with GitHub"
4. 授权Vercel访问你的GitHub账号
5. 完成账号设置

:::info 为什么选择Vercel
- 免费托管静态网站
- 自动部署功能
- 优秀的性能和可靠性
- 简单易用的界面
:::

</div>
