# 系统准备

## 检查电脑系统

### 系统要求
<div className="system-check">

1. 按下`Windows徽标键 + R`
2. 输入`msinfo32`并回车
3. 查看"系统类型"是64位还是32位
4. 确保系统是Windows 10或更高版本

:::tip 系统建议
- 推荐使用64位Windows 10或Windows 11系统
- 建议系统保持最新更新
- 确保有足够的磁盘空间（建议至少20GB）
:::

</div>

## 网络环境准备

### 科学上网配置
- 部分网站和工具需要科学上网环境
- 建议提前准备好稳定的梯子工具
- 确保能够访问GitHub和Vercel网站

:::caution 注意事项
请确保使用合法合规的网络工具，并遵守相关法律法规。
:::
