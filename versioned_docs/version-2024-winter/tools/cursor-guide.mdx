# Cursor使用指南

## Cursor简介

Cursor是一款革新性的代码编辑器，它集成了强大的AI功能，旨在提高开发者的工作效率。

- 智能代码生成

- 解释复杂代码
- 提供重构建议

- 支持主流编程语言

- 内置终端
- Git集成
- 调试工具

## 安装和设置

### 下载和安装步骤

<div className="step-by-step">

#### 1. 下载安装包

访问 [Cursor官网](https://cursor.sh/) 并点击"Download"按钮。

#### 2. 运行安装程序

根据你的操作系统选择合适的版本：

<Tabs>
  <TabItem value="windows" label="Windows" default>
    ```bash
    # 运行下载的.exe文件
    cursor-setup.exe
    ```
  </TabItem>
  <TabItem value="mac" label="macOS">
    ```bash
    # 打开.dmg文件并拖动到Applications文件夹
    # 或使用Homebrew安装
    brew install --cask cursor
    ```
  </TabItem>
  <TabItem value="linux" label="Linux">
    ```bash
    # 下载.AppImage文件
    chmod +x cursor.AppImage
    ./cursor.AppImage
    ```
  </TabItem>
</Tabs>

#### 3. 初始配置

1. 首次运行时登录或创建账号
2. 选择你喜欢的主题和字体
3. 配置常用编程语言
4. 设置AI功能（可能需要API密钥）

</div>

## 与AI协作开发

### 代码补全和建议

<div className="code-example">

#### 自动代码补全示例

```python
def calculate_average(numbers):
    # 开始输入'return'，AI会提供完整的平均值计算代码
    return sum(numbers) / len(numbers)
```

:::note 使用技巧
- 使用Tab键或Enter键接受建议
- 使用上下箭头键在多个建议间切换
- 按Esc键取消建议
:::

</div>

### 代码解释和重构

<Tabs>
  <TabItem value="original" label="原始代码">
    ```python
    def fibonacci(n):
        if n <= 1:
            return n
        else:
            return fibonacci(n-1) + fibonacci(n-2)
    ```
  </TabItem>
  <TabItem value="explanation" label="AI解释">
    ```
    这是一个递归实现的斐波那契数列函数：
    1. 基本情况：当n<=1时直接返回n
    2. 递归情况：返回前两个数的和
    3. 注意：这种实现方式时间复杂度为O(2^n)
    ```
  </TabItem>
  <TabItem value="optimized" label="AI优化建议">
    ```python
    def fibonacci(n):
        if n <= 1:
            return n
        a, b = 0, 1
        for _ in range(n):
            a, b = b, a + b
        return a
    ```
  </TabItem>
</Tabs>

### 错误诊断和修复

<div className="error-example">

#### 示例：语法错误修复

```python
# 有语法错误的代码
def greet(name)
    print("Hello, " + name + "!")

# Cursor会检测到缺少冒号，并提供修复建议：
def greet(name):
    print("Hello, " + name + "!")
```

</div>

### 使用自然语言生成代码

<div className="nlp-example">

#### 示例：通过注释生成代码

```python
# 创建一个函数，接受一个字符串列表，返回所有长度大于5的字符串
```

按下 `Ctrl + K`，AI会生成：

```python
def filter_long_strings(string_list):
    return [s for s in string_list if len(s) > 5]
```

</div>
