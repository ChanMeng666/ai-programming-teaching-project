<Admonition type="tip" title="最佳实践">
**全局 vs 项目级安装原则**：
- **全局安装**：系统工具、CLI 工具（如 Node.js、Git、Python）
- **项目级安装**：所有项目依赖、库、框架
- **使用虚拟环境**：Python 项目务必使用 venv，Node.js 项目依赖自动隔离
</Admonition>
