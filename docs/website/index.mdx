# 开发并部署个人网站

:::info 课程简介
本课程将带您使用 Gemini CLI 快速开发和部署一个专业的个人网站。从环境配置到网站上线，全程使用 AI 辅助，让网站开发变得简单高效。
:::

## 学习目标

完成本课程后，您将能够：

- **项目搭建**：使用 Vercel 模板快速启动项目，通过 Gemini CLI 配置开发环境
- **内容定制**：利用 AI 进行设计优化，打造独特的个人品牌展示
- **一键部署**：掌握现代化的部署流程，让网站快速上线
- **持续优化**：学会使用 AI 持续改进网站性能和用户体验

## 为什么需要个人网站

- **专业形象展示**：一个精心设计的个人网站是您专业能力的最佳证明，让您在求职和商务合作中脱颖而出
- **个人品牌建设**：通过网站展示您的独特价值，建立可识别的个人品牌，增强行业影响力
- **作品集中展示**：将您的项目、文章、成就集中展示，方便他人快速了解您的能力和经历
- **网络连接中心**：整合所有社交媒体和专业平台链接，成为您在线身份的统一入口

<Admonition type="tip" title="最佳实践">
**全局 vs 项目级安装原则**

- **全局安装**：系统工具、CLI 工具（如 Node.js、Git、Python）
- **项目级安装**：所有项目依赖、库、框架
- **使用虚拟环境**：Python 项目务必使用 venv，Node.js 项目依赖自动隔离
</Admonition>

## 选择心仪模板

### Vercel 模板库探索

前往 [https://vercel.com/templates](https://vercel.com/templates) 探索丰富的模板选择。

**模板分类**

- **Portfolio 作品集**：专为展示个人作品设计，适合设计师、开发者
- **Blog 博客**：内容为主的模板，适合写作者和思想分享者
- **Landing Page**：单页面展示，适合产品推广或个人介绍
- **Business 商务**：专业商务风格，适合咨询顾问或自由职业者

### 推荐模板：Magic Portfolio

**特点**

- 现代化设计，响应式布局
- 流畅的动画效果
- SEO 优化
- 易于定制

**技术栈**

- Next.js 14
- Tailwind CSS
- Framer Motion
- TypeScript 支持

访问模板页面：[Magic Portfolio 模板](https://vercel.com/templates/portfolio/magic-portfolio-for-next-js)。在页面中点击 **Deploy** 按钮开始部署。

## 项目部署流程

### 步骤 1：部署到 Vercel

1. **点击 Deploy 按钮**：在模板页面点击部署按钮，开始创建您的项目
2. **命名您的仓库**：使用您的名字命名，如 `chanmeng-portfolio`，这样域名会更个性化
3. **等待初始部署**：Vercel 会自动构建并部署您的网站，通常只需 1-2 分钟
