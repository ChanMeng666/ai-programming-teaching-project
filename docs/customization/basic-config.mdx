# 基础项目配置

## 修改网站基本信息

<div className="config-section">

### 配置网站信息
打开`docusaurus.config.js`文件，修改以下配置：

```javascript
module.exports = {
  // 网站标题
  title: '你的博客名称',
  // 标语
  tagline: '博客的简短描述',
  // 网站URL
  url: 'https://你的域名.com',
  // 网站图标
  favicon: 'img/favicon.ico',
  // 组织名称（通常是你的GitHub用户名）
  organizationName: '你的GitHub用户名',
  // 项目名称（将作为GitHub仓库名）
  projectName: '仓库名称',
}
```

:::tip 配置建议
- 选择一个简洁明了的网站标题
- 使用有吸引力的标语
- 确保URL配置正确
- 使用高质量的favicon
:::

</div>

## 自定义主题配置

### 导航栏配置

<div className="navbar-config">

```javascript
themeConfig: {
  navbar: {
    title: '网站标题',
    logo: {
      alt: 'Logo',
      src: 'img/logo.svg',
    },
    items: [
      {
        to: 'docs/',
        activeBasePath: 'docs',
        label: '文档',
        position: 'left',
      },
      {to: 'blog', label: '博客', position: 'left'},
      {
        href: 'https://github.com/你的用户名/仓库名',
        label: 'GitHub',
        position: 'right',
      },
    ],
  },
}
```

</div>

### 颜色主题配置

<div className="theme-config">

```javascript
module.exports = {
  themeConfig: {
    // 配置深色模式
    colorMode: {
      defaultMode: 'light',
      disableSwitch: false,
      respectPrefersColorScheme: true,
    },
  },
}
```

<Tabs>
  <TabItem value="light" label="浅色主题" default>
    - 默认主题模式
    - 适合日间阅读
    - 突出内容层次
  </TabItem>
  <TabItem value="dark" label="深色主题">
    - 护眼模式
    - 适合夜间阅读
    - 降低屏幕亮度
  </TabItem>
</Tabs>

</div>
