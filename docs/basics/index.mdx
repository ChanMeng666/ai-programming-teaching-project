## 📋 必备账号注册

在开始使用 Gemini CLI 之前，我们需要准备三个重要的账号。这些账号将贯穿整个开发流程。

<Tabs>
  <TabItem value="github" label="GitHub 账号" default>

### 为什么需要 GitHub？

GitHub 是全球最大的代码托管平台，我们将用它来：
- 存储和管理代码
- 与他人协作开发
- 展示个人项目作品集

### 注册步骤

1. 访问 [GitHub.com](https://github.com)
2. 点击右上角的 "Sign up"
3. 填写注册信息：
   - **Username**：选择一个专业的用户名（建议使用真实姓名或昵称）
   - **Email**：使用常用邮箱
   - **Password**：设置强密码

<Admonition type="tip" title="专业建议">
选择用户名时要慎重，因为它将出现在您所有的项目 URL 中。建议使用：
- 真实姓名：`john-smith`
- 专业昵称：`webdev-pro`
- 避免使用数字结尾：~~`coder123`~~
</Admonition>

### 初始配置

注册完成后，建议完成以下配置：

<CodeBlock language="bash">
{`# 配置 Git 全局用户信息
git config --global user.name "您的姓名"
git config --global user.email "your-email@example.com"`}
</CodeBlock>

### 安装步骤

<div className="installation-steps">

#### 步骤 1：检查 Node.js 版本

<CodeBlock language="bash">
{`# 检查 Node.js 是否已安装
node --version

# 如果未安装或版本过低，请访问 nodejs.org 下载安装`}
</CodeBlock>

#### 步骤 2：全局安装 Gemini CLI

<CodeBlock language="bash">
{`# 使用 npm 安装（推荐）
npm install -g @google/gemini-cli

# 或使用 yarn
yarn global add @google/gemini-cli

# 或使用 pnpm
pnpm add -g @google/gemini-cli`}
</CodeBlock>

<Admonition type="info" title="安装提示">
如果遇到权限问题，请：
- **Windows**：以管理员身份运行命令提示符
- **macOS/Linux**：使用 `sudo npm install -g @google/gemini-cli`
</Admonition>

#### 步骤 3：验证安装

<CodeBlock language="bash">
{`# 检查安装是否成功
gemini --version

# 查看可用命令
gemini --help`}
</CodeBlock>

</div>

### 首次配置

安装完成后，需要进行初始配置：

<CodeBlock language="bash">
{`# 1. 登录 Google 账号
gemini auth login

# 2. 系统将打开浏览器，请登录您的 Google 账号
# 3. 授权 Gemini CLI 访问必要的权限
# 4. 返回终端，您应该看到 "Successfully authenticated!" 的提示`}
</CodeBlock>

### 配置选项

<Tabs>
  <TabItem value="basic" label="基础配置" default>

<CodeBlock language="bash">
{`# 设置默认项目路径
gemini config set default-path ~/projects

# 设置语言偏好
gemini config set language zh-CN

# 查看当前配置
gemini config list`}
</CodeBlock>

  </TabItem>
  <TabItem value="advanced" label="高级配置">

<CodeBlock language="bash">
{`# 设置 API 超时时间（毫秒）
gemini config set timeout 30000

# 启用调试模式
gemini config set debug true

# 设置代理（如需要）
gemini config set proxy http://proxy.example.com:8080

# 导出配置文件
gemini config export > gemini-config.json`}
</CodeBlock>

  </TabItem>
</Tabs>

## 🚫 告别繁琐的传统方式

### 不再需要的工具

<div className="comparison-table">

| 传统工具 | 被 Gemini CLI 替代的原因 |
|---------|------------------------|
| **Cursor** | • 中国大陆锁区<br/>• Pro 额度性价比低<br/>• Gemini CLI 完全免费可用 |
| **GitHub Desktop** | • 无需 GUI 工具<br/>• Gemini 可以执行所有 Git 操作<br/>• 更灵活的命令行交互 |
| **Git 命令** | • 不用记忆任何 Git 命令<br/>• 自然语言描述即可<br/>• AI 自动处理复杂操作 |

</div>
