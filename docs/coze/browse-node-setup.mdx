## ⚙️ 节点配置

<Tabs>
  <TabItem value="basic" label="基本设置" default>
    <div className="step-content">
      <h3>1️⃣ 添加节点</h3>
      <ol>
        <li>在工作流编辑器中点击"+"按钮</li>
        <li>选择"Browse"节点类型</li>
        <li>将节点连接到Start节点</li>
      </ol>
    </div>
  </TabItem>
  <TabItem value="params" label="参数配置">
    <div className="step-content">
      <h3>2️⃣ 设置参数</h3>
      
      ```typescript
      // url参数配置
      {
        "url": "{{input}}",
        "enforce_crawl": true
      }
      ```
      
      <Admonition type="tip" title="参数说明">
        <ul>
          <li><code>url</code>: 使用变量引用输入的URL</li>
          <li><code>enforce_crawl</code>: 设为true确保完整抓取</li>
        </ul>
      </Admonition>
    </div>
  </TabItem>
  <TabItem value="output" label="输出说明">
    <div className="step-content">
      <h3>3️⃣ 输出结构</h3>
      
      ```json
      {
        "code": 200,
        "message": "success",
        "plugin_observation": {
          "observation": "网页实际内容",
          "natural_language_desc": "内容描述信息"
        }
      }
      ```
    </div>
  </TabItem>
</Tabs>
