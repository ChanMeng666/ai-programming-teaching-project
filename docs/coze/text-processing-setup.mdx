# 文本处理与存储

## Text Processing配置

### 节点设置
1. 添加Text Processing节点
2. 连接到LLM节点

### 参数配置
1. 输入设置：
```typescript
String1: {{llm.output.output}}
```

2. 格式模板：
```text
=========================================
              文章摘要
=========================================

{{String1}}
=========================================
```

## Notion集成

### Write Page节点配置
1. 添加Write Page节点
2. 连接到Text Processing节点

### 参数设置
```typescript
page_id: "你的Notion页面ID"
text: {{Text Processing.output}}
```
