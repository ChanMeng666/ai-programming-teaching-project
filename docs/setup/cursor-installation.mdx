# 安装Cursor编辑器

## 下载Cursor

<div className="download-steps">

### 1. 获取安装程序
1. 访问官网：https://cursor.sh/
2. 点击"Download"按钮下载安装程序
3. 等待下载完成

### 2. 安装过程
<Tabs>
  <TabItem value="windows" label="Windows" default>
    1. 双击下载的安装程序
    2. 点击"Next"继续安装
    3. 选择安装位置（建议保持默认）
    4. 等待安装完成
    5. 点击"Finish"完成安装
  </TabItem>
  <TabItem value="mac" label="macOS">
    1. 打开下载的.dmg文件
    2. 将Cursor拖入Applications文件夹
    3. 首次运行时点击"打开"确认
  </TabItem>
</Tabs>

### 3. 初始化设置
1. 从开始菜单或桌面启动Cursor
2. 会弹出欢迎界面
3. 选择喜欢的主题颜色（深色或浅色）
4. 完成初始化设置

</div>

:::tip 安装建议
- 建议使用管理员权限安装
- 如果安装过程中遇到防火墙提示，请允许访问
- 安装完成后建议重启电脑
:::
